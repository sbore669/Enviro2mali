== Description ==

Planet Green is child theme of Bizberg WordPress Theme. Theme is a clean & modern Eco Nature WordPress Theme. Planet Green is suitable for any nature, environment, environmental, donation, action, green, animal, africa, wild, non profit, foundation, paypal, natural, ngo, earth websites. This Free WordPress theme is fully responsive, cross-browser compatible, translation ready & SEO friendly. If you run into any problem while using our theme, you can refer to extensive documentation or contact our friendly support team. It fully supports Elementor, all the sections are made from Elementor.
